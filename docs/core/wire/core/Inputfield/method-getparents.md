# $inputfield->getParents(): array

Source: `wire/core/Inputfield.php`

Get array of all parents of this Inputfield.

## Usage

~~~~~
// basic usage
$array = $inputfield->getParents();
~~~~~

## Return value

- `array` of InputfieldWrapper elements.
