# $inputfield->setParent(InputfieldWrapper $parent): $this

Source: `wire/core/Inputfield.php`

Set the parent (InputfieldWrapper) of this Inputfield.

## Usage

~~~~~
// basic usage
$result = $inputfield->setParent($parent);

// usage with all arguments
$result = $inputfield->setParent(InputfieldWrapper $parent);
~~~~~

## Arguments

- `$parent` `InputfieldWrapper`

## Return value

- `$this`
