# WireArray::__callStatic($name, $arguments): WireArray

Source: `wire/core/WireArray.php`

Static method caller, primarily for support of WireArray::new() method

## Usage

~~~~~
// basic usage
$items = WireArray::__callStatic($name, $arguments);
~~~~~

## Arguments

- `$name` `string`
- `$arguments` `array`

## Return value

- `WireArray`

## Exceptions

- `WireException`
