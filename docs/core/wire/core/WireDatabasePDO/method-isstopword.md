# $wireDatabasePDO->isStopword($word, $engine = ''): bool

Source: `wire/core/WireDatabasePDO.php`

Is given word a fulltext stopword for database engine?

- string $word
- string $engine DB engine ('myisam' or 'innodb') or omit for current engine
