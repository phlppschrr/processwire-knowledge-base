# $wire->untrackChange($what): $this

Source: `wire/core/Wire.php`

Untrack a change to a property in this object

## Usage

~~~~~
// basic usage
$result = $wire->untrackChange($what);
~~~~~

## Arguments

- `$what` `string` Name of property that you want to remove its change being tracked

## Return value

- `$this`
