# $pagesPathFinder->getBlankResult(array $result = array()): array

Source: `wire/core/PagesPathFinder.php`

Build blank result/return value array

## Usage

~~~~~
// basic usage
$array = $pagesPathFinder->getBlankResult();

// usage with all arguments
$array = $pagesPathFinder->getBlankResult(array $result = array());
~~~~~

## Arguments

- `$result` (optional) `array` Optionally return blank result merged with this one

## Return value

- `array`
