# $fieldgroups->deleteField(Field $field): bool

Source: `wire/core/Fieldgroups.php`

Delete the entries in fieldgroups_fields for the given Field

## Usage

~~~~~
// basic usage
$bool = $fieldgroups->deleteField($field);

// usage with all arguments
$bool = $fieldgroups->deleteField(Field $field);
~~~~~

## Arguments

- `$field` `Field`

## Return value

- `bool`
