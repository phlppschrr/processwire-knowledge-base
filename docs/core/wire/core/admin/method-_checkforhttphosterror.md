# $admin->_checkForHttpHostError(Config $config)

Source: `wire/core/admin.php`

Check if the current HTTP host is recognized and generate error if not

## Usage

~~~~~
// basic usage
$result = $admin->_checkForHttpHostError($config);

// usage with all arguments
$result = $admin->_checkForHttpHostError(Config $config);
~~~~~

## Arguments

- `$config` `Config`
