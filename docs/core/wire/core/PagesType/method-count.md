# $pagesType->count($selectorString = '', array $options = array()): int

Source: `wire/core/PagesType.php`

Return the number of pages in this type matching the given selector string

## Arguments

- `$selectorString` (optional) `string` Optional, if omitted then returns count of all pages of this type
- `$options` (optional) `array` Options to modify default behavior (see $pages->count method for details)

## Return value

int
