# $wireCache->setCacheModule(WireCacheInterface $module)

Source: `wire/core/WireCache.php`

Set WireCache module to use for caching

## Usage

~~~~~
// basic usage
$result = $wireCache->setCacheModule($module);

// usage with all arguments
$result = $wireCache->setCacheModule(WireCacheInterface $module);
~~~~~

## Arguments

- `$module` `WireCacheInterface`
