# $modulesLoader->readyModule(Module $module): bool

Source: `wire/core/ModulesLoader.php`

Call ready for a single module

## Usage

~~~~~
// basic usage
$bool = $modulesLoader->readyModule($module);

// usage with all arguments
$bool = $modulesLoader->readyModule(Module $module);
~~~~~

## Arguments

- `$module` `Module`

## Return value

- `bool`
