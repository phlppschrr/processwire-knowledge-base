# $imageSizerEngineGD->prepareImageLayer(&$im, &$image)

Source: `wire/core/ImageSizerEngineGD.php`

Prepares a new created GD image resource according to the IMAGETYPE

Intended for use by the resize() method

## Arguments

- `$im` `resource` , destination resource needs to be prepared
- `$image` `resource` , with GIF we need to read from source resource
