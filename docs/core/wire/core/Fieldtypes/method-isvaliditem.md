# $fieldtypes->isValidItem($item): bool

Source: `wire/core/Fieldtypes.php`

Per WireArray interface, items added to Fieldtypes must be Fieldtype instances

## Usage

~~~~~
// basic usage
$bool = $fieldtypes->isValidItem($item);
~~~~~

## Arguments

- `$item` `Wire|Fieldtype`

## Return value

- `bool`
