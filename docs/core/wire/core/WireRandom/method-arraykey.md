# $wireRandom->arrayKey(array $a): string|int

Source: `wire/core/WireRandom.php`

Get a random key from given array

## Usage

~~~~~
// basic usage
$string = $wireRandom->arrayKey($a);

// usage with all arguments
$string = $wireRandom->arrayKey(array $a);
~~~~~

## Arguments

- `$a` `array`

## Return value

- `string|int`
