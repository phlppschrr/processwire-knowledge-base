# $wireRandom->arrayValue(array $a): mixed|null

Source: `wire/core/WireRandom.php`

Get a random value from given array

## Usage

~~~~~
// basic usage
$result = $wireRandom->arrayValue($a);

// usage with all arguments
$result = $wireRandom->arrayValue(array $a);
~~~~~

## Arguments

- `$a` `array` Array to get random value from

## Return value

- `mixed|null`
