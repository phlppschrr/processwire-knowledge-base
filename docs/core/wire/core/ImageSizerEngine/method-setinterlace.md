# $imageSizerEngine->setInterlace($value = true): self

Source: `wire/core/ImageSizerEngine.php`

Turn on/off interlace

## Usage

~~~~~
// basic usage
$result = $imageSizerEngine->setInterlace();

// usage with all arguments
$result = $imageSizerEngine->setInterlace($value = true);
~~~~~

## Arguments

- `$value` (optional) `bool` Whether to upscale or not (default = true)

## Return value

- `self`
