# $field->get($key): mixed

Source: `wire/core/Field.php`

Get a Field setting or dynamic data property

This can also be accessed directly, i.e. `$fieldName = $field->name;`.

## Usage

~~~~~
// basic usage
$result = $field->get($key);
~~~~~

## Arguments

- `$key` `string`

## Return value

- `mixed`
