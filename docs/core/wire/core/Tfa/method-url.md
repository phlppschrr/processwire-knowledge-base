# $tfa->url($queryString = ''): string

Source: `wire/core/Tfa.php`

Get the start URL and optionally append query string

## Usage

~~~~~
// basic usage
$string = $tfa->url();

// usage with all arguments
$string = $tfa->url($queryString = '');
~~~~~

## Arguments

- `$queryString` (optional) `string`

## Return value

- `string`
