# $tfa->getTfaTypeName(): string

Source: `wire/core/Tfa.php`

Get translated Tfa type name (short name)

When a module implements this, it should not make a a parent::getTfaTypeName() call.

## Usage

~~~~~
// basic usage
$string = $tfa->getTfaTypeName();
~~~~~

## Return value

- `string`
