# $adminThemeFramework->includeFile($basename, array $vars = array())

Source: `wire/core/AdminThemeFramework.php`

Include an admin theme file

## Usage

~~~~~
// basic usage
$result = $adminThemeFramework->includeFile($basename);

// usage with all arguments
$result = $adminThemeFramework->includeFile($basename, array $vars = array());
~~~~~

## Arguments

- `$basename` `string`
- `$vars` (optional) `array`
