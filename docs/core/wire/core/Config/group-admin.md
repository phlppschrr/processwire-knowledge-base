# Config: admin

Source: `wire/core/Config.php`

- $logs: array Additional core logs to keep

- $logIP: bool Include IP address in logs, when applicable?

- $defaultAdminTheme: string Default admin theme: AdminThemeUikit, AdminThemeDefault or AdminThemeReno.

- $AdminThemeUikit: array Settings specific to AdminThemeUikit module (see this setting in /wire/config.php). @since 3.0.179

- $modals: array Settings for modal windows
