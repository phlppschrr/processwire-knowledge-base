# Config: images

Source: `wire/core/Config.php`

- [$imageSizes: array](method-imagesizes.md) Predefined image sizes (and options) indexed by name. See /wire/config.php for example. @since 3.0.151
- [$imageSizerOptions: array](method-imagesizeroptions.md) Options to set image sizing defaults. Please see the /wire/config.php file for all options and defaults.
- [$webpOptions: array](method-webpoptions.md) Options for webp images. Please see /wire/config.php for all options.
- $adminThumbOptions: array Admin thumbnail image options
