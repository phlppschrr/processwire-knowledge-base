# Config: images

Source: `wire/core/Config.php`

- $imageSizes: array Predefined image sizes (and options) indexed by name. See /wire/config.php for example. @since 3.0.151

- $imageSizerOptions: array Options to set image sizing defaults. Please see the /wire/config.php file for all options and defaults.

- $webpOptions: array Options for webp images. Please see /wire/config.php for all options.

- $adminThumbOptions: array Admin thumbnail image options
