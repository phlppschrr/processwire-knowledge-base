# $pagefile->filesize($reset = false): int

Source: `wire/core/Pagefile.php`

Returns the filesize in number of bytes.

## Usage

~~~~~
// basic usage
$int = $pagefile->filesize();

// usage with all arguments
$int = $pagefile->filesize($reset = false);
~~~~~

## Arguments

- `$reset` (optional) `bool`

## Return value

- `int`
