# Template: other

Source: `wire/core/Template.php`

- $modified: int Last modified time for template or template file
- $compile: int Set to 1 to enable compilation, 2 to compile file and included files, 3 for auto, or 0 to disable.
- $pageLabelField: string CSV or space separated string of field names to be displayed by ProcessPageList (overrides those set with ProcessPageList config).
