# $wireTextTools->getPunctuationChars($sentence = false): array

Source: `wire/core/WireTextTools.php`

Get array of punctuation characters

## Usage

~~~~~
// basic usage
$array = $wireTextTools->getPunctuationChars();

// usage with all arguments
$array = $wireTextTools->getPunctuationChars($sentence = false);
~~~~~

## Arguments

- `$sentence` (optional) `bool` Get only sentence-ending punctuation

## Return value

- `array`
