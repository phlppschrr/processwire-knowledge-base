# $pages->___trashReady(Page $page)

Source: `wire/core/Pages.php`

Hook called when a Page is about to be trashed

## Usage

~~~~~
// basic usage
$result = $pages->___trashReady($page);

// usage with all arguments
$result = $pages->___trashReady(Page $page);
~~~~~

## Arguments

- `$page` `Page`
