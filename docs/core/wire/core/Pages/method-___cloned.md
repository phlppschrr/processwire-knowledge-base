# $pages->___cloned(Page $page, Page $copy)

Source: `wire/core/Pages.php`

Hook called when a page has been cloned

## Usage

~~~~~
// basic usage
$result = $pages->___cloned($page, $copy);

// usage with all arguments
$result = $pages->___cloned(Page $page, Page $copy);
~~~~~

## Arguments

- `$page` `Page` The original page to be cloned
- `$copy` `Page` The completed cloned version of the page
