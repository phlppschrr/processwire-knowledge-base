# $notification->removeFlag($name): self

Source: `wire/modules/System/SystemNotifications/Notification.php`

Remove the given flag name(s) (shortcut for setFlag)

## Usage

~~~~~
// basic usage
$result = $notification->removeFlag($name);
~~~~~

## Arguments

- `$name` `string` One or more space-separated flag names

## Return value

- `self`
