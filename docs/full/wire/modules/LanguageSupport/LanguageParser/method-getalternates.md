# $languageParser->getAlternates($hash = ''): array

Source: `wire/modules/LanguageSupport/LanguageParser.php`

Get phrase alternates

## Usage

~~~~~
// basic usage
$array = $languageParser->getAlternates();

// usage with all arguments
$array = $languageParser->getAlternates($hash = '');
~~~~~

## Arguments

- `$hash` (optional) `string` Specify phrase hash to get alternates or omit to get all alternates

## Return value

- `array`
