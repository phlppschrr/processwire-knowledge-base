# $languages->importTranslationsFile($language, $file, $quiet = false): bool|int

Source: `wire/modules/LanguageSupport/Languages.php`

Import a language translations file

## Usage

~~~~~
// basic usage
$bool = $languages->importTranslationsFile($language, $file);

// usage with all arguments
$bool = $languages->importTranslationsFile($language, $file, $quiet = false);
~~~~~

## Arguments

- `$language` `Language|string`
- `$file` `string` Full path to .csv translations file The .csv file must be one generated by ProcessWireâ€™s language translation tools.
- `$quiet` (optional) `bool` Specify true to suppress error/success notifications being generated (default=false)

## Return value

- `bool|int` Returns integer with number of translations imported or boolean false on error

## Exceptions

- `WireException`
