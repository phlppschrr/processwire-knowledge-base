# $processPageSearchLive->setNoSearchTypes(array $types)

Source: `wire/modules/Process/ProcessPageSearch/ProcessPageSearchLive.php`

Set types that should be excluded unless specifically asked for

## Usage

~~~~~
// basic usage
$result = $processPageSearchLive->setNoSearchTypes($types);

// usage with all arguments
$result = $processPageSearchLive->setNoSearchTypes(array $types);
~~~~~

## Arguments

- `$types` `array` Names of types to exclude
