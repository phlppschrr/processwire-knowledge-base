# $fieldsetPage->getUnformatted($key): mixed

Source: `wire/modules/Fieldtype/FieldtypeRepeater/FieldsetPage.php`

Get the unformatted value of a field, regardless of current output formatting state

## Usage

~~~~~
// basic usage
$result = $fieldsetPage->getUnformatted($key);
~~~~~

## Arguments

- `$key` `string` Field or property name to retrieve

## Return value

- `mixed`
