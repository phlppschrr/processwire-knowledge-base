# $commentFormCustom->applyIf($name, $value, $out): mixed|string

Source: `wire/modules/Fieldtype/FieldtypeComments/CommentFormCustom.php`

Apply an {if.name} ... {endif.name} statement

## Arguments

- `$name` `string` Property name for if statement
- `$value` `bool` If true the contents of the {if...} statement will stay, if false it will be removed
- `$out` `string`

## Return value

mixed|string
