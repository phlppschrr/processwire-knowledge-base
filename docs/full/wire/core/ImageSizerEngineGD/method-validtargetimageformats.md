# $imageSizerEngineGD->validTargetImageFormats(): array

Source: `wire/core/ImageSizerEngineGD.php`

Get an array of image file extensions this ImageSizerModule can create

## Usage

~~~~~
// basic usage
$array = $imageSizerEngineGD->validTargetImageFormats();
~~~~~

## Return value

- `array` of uppercase file extensions, i.e. ['PNG', 'JPG']
