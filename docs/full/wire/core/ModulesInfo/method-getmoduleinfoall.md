# $modulesInfo->getModuleInfoAll(array $options = array()): array

Source: `wire/core/ModulesInfo.php`

Get info arrays for all modules indexed by module name

## Usage

~~~~~
// basic usage
$array = $modulesInfo->getModuleInfoAll();

// usage with all arguments
$array = $modulesInfo->getModuleInfoAll(array $options = array());
~~~~~

## Arguments

- `$options` (optional) `array` See options for getModuleInfo() method

## Return value

- `array`
