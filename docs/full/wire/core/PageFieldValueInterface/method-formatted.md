# $pageFieldValueInterface->formatted($set = null): bool

Source: `wire/core/Interfaces.php`

Get or set formatted state

## Usage

~~~~~
// basic usage
$bool = $pageFieldValueInterface->formatted();

// usage with all arguments
$bool = $pageFieldValueInterface->formatted($set = null);
~~~~~

## Arguments

- `$set` (optional) `bool|null` Specify bool to set formatted state or omit to retrieve formatted state

## Return value

- `bool`
