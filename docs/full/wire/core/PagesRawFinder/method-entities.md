# $pagesRawFinder->entities(&$value)

Source: `wire/core/PagesRaw.php`

Apply entity encoding to all strings in given value, recursively

## Usage

~~~~~
// basic usage
$result = $pagesRawFinder->entities($value);

// usage with all arguments
$result = $pagesRawFinder->entities(&$value);
~~~~~

## Arguments

- `$value` `mixed`
