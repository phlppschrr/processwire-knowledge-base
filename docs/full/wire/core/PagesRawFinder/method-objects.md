# $pagesRawFinder->objects(&$values)

Source: `wire/core/PagesRaw.php`

Convert associative arrays to objects

## Usage

~~~~~
// basic usage
$result = $pagesRawFinder->objects($values);

// usage with all arguments
$result = $pagesRawFinder->objects(&$values);
~~~~~

## Arguments

- `$values` `array`
