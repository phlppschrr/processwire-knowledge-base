# $inputfield->wrapClass($class = null): string|Inputfield

Source: `wire/core/Inputfield.php`

Get wrapper class attribute or add a class to it. @since 3.0.110

## Usage

~~~~~
// basic usage
$string = $inputfield->wrapClass();

// usage with all arguments
$string = $inputfield->wrapClass($class = null);
~~~~~
