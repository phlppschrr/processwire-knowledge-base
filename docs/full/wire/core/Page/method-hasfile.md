# $page->hasFile($file, array $options = array()): bool|string

Source: `wire/core/Page.php`

Does Page have given filename in its files directory?

## Arguments

- `$file` `string` File basename or verbose hash
- `$options` (optional) `array` - `getPathname` (bool): Get full path + filename when would otherwise return boolean true? (default=false) - `getPagefile` (bool): Get Pagefile object when would otherwise return boolean true? (default=false)

## Return value

bool|string
