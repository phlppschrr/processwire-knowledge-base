# $selectors->matches(Wire $item): bool

Source: `wire/core/Selectors.php`

Does the given Wire match these Selectors?

## Usage

~~~~~
// basic usage
$bool = $selectors->matches($item);

// usage with all arguments
$bool = $selectors->matches(Wire $item);
~~~~~

## Arguments

- `$item` `Wire`

## Return value

- `bool`
