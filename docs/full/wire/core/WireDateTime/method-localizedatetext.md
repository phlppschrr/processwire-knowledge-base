# $wireDateTime->localizeDateText($value, $format): string

Source: `wire/core/WireDateTime.php`

Localize a date's month and day names, when present

## Usage

~~~~~
// basic usage
$string = $wireDateTime->localizeDateText($value, $format);
~~~~~

## Arguments

- `$value` `string` Date that is already formated with $format
- `$format` `string` Format that was used to format the date

## Return value

- `string`
