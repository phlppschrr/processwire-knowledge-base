# $pageTraversal->_nextFinderOptions(Page $page, $selector, $options): array

Source: `wire/core/PageTraversal.php`

Builds the PageFinder options for the _next() method

~~~~~
// basic usage

// usage with all arguments
$array = $pageTraversal->_nextFinderOptions(Page $page, $selector, $options);
~~~~~

## Arguments

- `$page` `Page`
- `$selector` `string|array|Selectors`
- `$options` `array`

## Return value

- `array`
