# $wireArray->prepend($item): $this

Source: `wire/core/WireArray.php`

Prepend an item to the beginning of the WireArray.

~~~~~
// Add item to beginning
$items->prepend($item);
~~~~~

## Arguments

- `$item` `Wire|WireArray|mixed` Item to prepend.

## Return value

$this This instance.

## Throws

- WireException
