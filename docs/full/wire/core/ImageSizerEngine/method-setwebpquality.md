# $imageSizerEngine->setWebpQuality($n): self

Source: `wire/core/ImageSizerEngine.php`

Set the image quality 1-100 for WebP output, where 100 is highest quality

## Usage

~~~~~
// basic usage
$result = $imageSizerEngine->setWebpQuality($n);
~~~~~

## Arguments

- `$n` `int`

## Return value

- `self`
