# $pagesRequest->setResponseCode($code, $message = '')

Source: `wire/core/PagesRequest.php`

Set response code and type

## Usage

~~~~~
// basic usage
$result = $pagesRequest->setResponseCode($code);

// usage with all arguments
$result = $pagesRequest->setResponseCode($code, $message = '');
~~~~~

## Arguments

- `$code` `int`
- `$message` (optional) `string` Optional message string
