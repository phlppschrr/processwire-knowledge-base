# $fileCompiler->clearCache($all = false): bool

Source: `wire/core/FileCompiler.php`

Clear all file compiler caches

## Usage

~~~~~
// basic usage
$bool = $fileCompiler->clearCache();

// usage with all arguments
$bool = $fileCompiler->clearCache($all = false);
~~~~~

## Arguments

- `$all` (optional) `bool` Specify true to clear for all FileCompiler caches

## Return value

- `bool`
