# $pageFinder->preProcessFieldtypeSelector(Selectors $selectors, Selector $selector)

Source: `wire/core/PageFinder.php`

Pre-process a selector having field name that begins with "Fieldtype"

## Usage

~~~~~
// basic usage
$result = $pageFinder->preProcessFieldtypeSelector($selectors, $selector);

// usage with all arguments
$result = $pageFinder->preProcessFieldtypeSelector(Selectors $selectors, Selector $selector);
~~~~~

## Arguments

- `$selectors` `Selectors`
- `$selector` `Selector`
