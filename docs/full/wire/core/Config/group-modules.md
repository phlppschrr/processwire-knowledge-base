# Config: modules

Source: `wire/core/Config.php`

- [$pageList: array](method-pagelist.md) Settings specific to Page lists.
- [$pageEdit: array](method-pageedit.md) Settings specific to Page editors.
- [$pageAdd: array](method-pageadd.md) Settings specific to Page adding.
- $moduleServiceURL: string URL where the modules web service can be accessed
- $moduleServiceKey: string API key for modules web service
- $moduleCompile: bool Allow use of compiled modules?
- [$moduleInstall: array](method-moduleinstall.md) Admin module install options you allow.
- [$substituteModules: array](method-substitutemodules.md) Associative array with names of substitute modules for when requested module doesn't exist
- [$markupQA: array](method-markupqa.md) Optional settings for MarkupQA class used by FieldtypeTextarea module.
