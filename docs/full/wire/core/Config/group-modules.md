# Config: modules

Source: `wire/core/Config.php`

- $pageList: array Settings specific to Page lists.

- $pageEdit: array Settings specific to Page editors.

- $pageAdd: array Settings specific to Page adding.

- $moduleServiceURL: string URL where the modules web service can be accessed

- $moduleServiceKey: string API key for modules web service

- $moduleCompile: bool Allow use of compiled modules?

- $moduleInstall: array Admin module install options you allow.

- $substituteModules: array Associative array with names of substitute modules for when requested module doesn't exist

- $markupQA: array Optional settings for MarkupQA class used by FieldtypeTextarea module.
