# HasLookupItems

Source: `wire/core/Interfaces.php`

For classes that contain lookup items, as used by WireSaveableItemsLookup

Methods:
- [`getLookupItems()`](method-getlookupitems.md) Get all lookup items, usually in a WireArray derived type, but specified by class
- [`addLookupItem(int $item, array &$row)`](method-addlookupitem.md) Add a lookup item to this instance
