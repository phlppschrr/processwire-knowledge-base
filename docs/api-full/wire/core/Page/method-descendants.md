# $page->descendants($selector = '', array $options = array()): PageArray

Source: `wire/core/Page.php`

Find descendant pages, alias of `Page::find()`, see that method for details. @since 3.0.116

## Usage

~~~~~
// basic usage
$items = $page->descendants();

// usage with all arguments
$items = $page->descendants($selector = '', array $options = array());
~~~~~
