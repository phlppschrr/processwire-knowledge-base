# $pageimageVariations->count($options = array()): int

Source: `wire/core/PageimageVariations.php`

Return a total or filtered count of variations

This method is also here to implement the Countable interface.

## Usage

~~~~~
// basic usage
$int = $pageimageVariations->count();

// usage with all arguments
$int = $pageimageVariations->count($options = array());
~~~~~

## Arguments

- `$options` (optional) `array` See options for find() method

## Return value

- `int`
