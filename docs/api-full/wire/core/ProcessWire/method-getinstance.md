# ProcessWire::getInstance($instanceID = null): null|ProcessWire

Source: `wire/core/ProcessWire.php`

Get a ProcessWire instance by ID

## Usage

~~~~~
// basic usage
ProcessWire::getInstance();

// usage with all arguments
ProcessWire::getInstance($instanceID = null);
~~~~~

## Arguments

- `$instanceID` (optional) `int|null` Omit this argument to return the current instance

## Return value

- `null|ProcessWire`
