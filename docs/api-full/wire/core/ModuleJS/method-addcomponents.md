# $moduleJS->addComponents(array $components): $this

Source: `wire/core/ModuleJS.php`

Add an array of optional components

## Usage

~~~~~
// basic usage
$result = $moduleJS->addComponents($components);

// usage with all arguments
$result = $moduleJS->addComponents(array $components);
~~~~~

## Arguments

- `$components` `array`

## Return value

- `$this`
