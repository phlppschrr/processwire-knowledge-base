# $wireSaveableItems->loadRowsReady(array &$rows)

Source: `wire/core/WireSaveableItems.php`

Called after rows loaded from DB but before populated to this instance

## Usage

~~~~~
// basic usage
$result = $wireSaveableItems->loadRowsReady($rows);

// usage with all arguments
$result = $wireSaveableItems->loadRowsReady(array &$rows);
~~~~~

## Arguments

- `$rows` `array`
