# $wireDatabasePDO->escapeStr($str): string

Source: `wire/core/WireDatabasePDO.php`

Escape a string value, same as $database->quote() but without surrounding quotes

## Usage

~~~~~
// basic usage
$string = $wireDatabasePDO->escapeStr($str);
~~~~~

## Arguments

- `$str` `string`

## Return value

- `string`
