# $fileCompiler->optionsToString(array $options): string

Source: `wire/core/FileCompiler.php`

Given an array of $options convert to an PHP-code array() string

## Usage

~~~~~
// basic usage
$string = $fileCompiler->optionsToString($options);

// usage with all arguments
$string = $fileCompiler->optionsToString(array $options);
~~~~~

## Arguments

- `$options` `array`

## Return value

- `string`
