# Selector: other

Source: `wire/core/Selector.php`

- [`$fields: array`](method-fields.md) Fields that were present in selector (same as `$field`, but always an array).
- [`$field: string|array`](method-field.md) Field or fields present in the selector (string if single, or array of strings if multiple). Preferable to use `$fields` property instead.
- [`$operator: string`](method-operator.md) Operator used by the selector.
- [`$values: array`](method-values.md) Values that were present in selector (same as `$value`, but always array).
- [`$value: string|array`](method-value.md) Value or values present in the selector (string if single, or array of strings if multiple). Preferable to use `$values` property instead.
