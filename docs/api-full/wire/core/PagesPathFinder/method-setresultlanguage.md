# $pagesPathFinder->setResultLanguage($language, $segment = '')

Source: `wire/core/PagesPathFinder.php`

Set result language by name or ID

## Usage

~~~~~
// basic usage
$result = $pagesPathFinder->setResultLanguage($language);

// usage with all arguments
$result = $pagesPathFinder->setResultLanguage($language, $segment = '');
~~~~~

## Arguments

- `$language` `int|string|Language`
- `$segment` (optional) `string`
