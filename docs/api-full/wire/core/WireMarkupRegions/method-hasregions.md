# $wireMarkupRegions->hasRegions(&$html): bool

Source: `wire/core/WireMarkupRegions.php`

## Usage

~~~~~
// basic usage
$bool = $wireMarkupRegions->hasRegions($html);

// usage with all arguments
$bool = $wireMarkupRegions->hasRegions(&$html);
~~~~~

## Arguments

- `$html` `string`

## Return value

- `bool`
