# $wireMarkupRegions->initHtml(&$html)

Source: `wire/core/WireMarkupRegions.php`

Initialize given HTML for markup regions

## Usage

~~~~~
// basic usage
$result = $wireMarkupRegions->initHtml($html);

// usage with all arguments
$result = $wireMarkupRegions->initHtml(&$html);
~~~~~

## Arguments

- `$html` `string`
