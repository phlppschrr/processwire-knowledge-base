# $wireMarkupRegions->stripOptional($markup, $debug = false): string

Source: `wire/core/WireMarkupRegions.php`

Strip optional tags/comments from given markup

## Usage

~~~~~
// basic usage
$string = $wireMarkupRegions->stripOptional($markup);

// usage with all arguments
$string = $wireMarkupRegions->stripOptional($markup, $debug = false);
~~~~~

## Arguments

- `$markup` `string`
- `$debug` (optional) `bool`

## Return value

- `string`
