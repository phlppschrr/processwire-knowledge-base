# $wireDateTime->dateWords($key): array

Source: `wire/core/WireDateTime.php`

Get translated month/day words indexed by English words

## Usage

~~~~~
// basic usage
$array = $wireDateTime->dateWords($key);
~~~~~

## Arguments

- `$key` `string` One of: monthNames, monthAbbrs, dayNames, dayAbbrs, meridiums

## Return value

- `array`
