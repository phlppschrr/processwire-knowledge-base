# $pageArray->getPageByID($id): NullPage|Page

Source: `wire/core/PageArray.php`

Get Page from this PageArray having given ID, or return NullPage if not present

## Usage

~~~~~
// basic usage
$nullPage = $pageArray->getPageByID($id);
~~~~~

## Arguments

- `$id` `int`

## Return value

- `NullPage|Page`

## Since

3.0.162
