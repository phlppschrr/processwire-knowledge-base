# $fieldtypeHasFiles->hasFiles(Page $page, Field $field): bool

Source: `wire/core/Interfaces.php`

// basic usage

// usage with all arguments
$bool = $fieldtypeHasFiles->hasFiles(Page $page, Field $field);
~~~~~

## Arguments

- `$page` `Page`
- `$field` `Field`

## Return value

- `bool`
