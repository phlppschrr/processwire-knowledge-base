# $imageSizerEngine->setUpscaling($value = true): self

Source: `wire/core/ImageSizerEngine.php`

Turn on/off upscaling

## Usage

~~~~~
// basic usage
$result = $imageSizerEngine->setUpscaling();

// usage with all arguments
$result = $imageSizerEngine->setUpscaling($value = true);
~~~~~

## Arguments

- `$value` (optional) `bool` Whether to upscale or not (default = true)

## Return value

- `self`
