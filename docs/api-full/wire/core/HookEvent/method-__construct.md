# $hookEvent->__construct(array $eventData = array())

Source: `wire/core/HookEvent.php`

Construct the HookEvent and establish default values

## Usage

~~~~~
// basic usage
$result = $hookEvent->__construct();

// usage with all arguments
$result = $hookEvent->__construct(array $eventData = array());
~~~~~

## Arguments

- `$eventData` (optional) `array` Optional event data to start with
