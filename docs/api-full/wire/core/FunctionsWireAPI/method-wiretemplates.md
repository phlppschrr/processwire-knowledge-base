# $functionsWireAPI->wireTemplates($name = ''): Templates|Template|null

Source: `wire/core/FunctionsWireAPI.php`

Access the $templates API variable as a function

## Usage

~~~~~
// basic usage
$templates = $functionsWireAPI->wireTemplates();

// usage with all arguments
$templates = $functionsWireAPI->wireTemplates($name = '');
~~~~~

## Arguments

- `$name` (optional) `string` Optional template to retrieve

## Return value

- `Templates|Template|null`
