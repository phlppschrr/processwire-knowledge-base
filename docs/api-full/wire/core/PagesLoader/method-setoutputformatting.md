# $pagesLoader->setOutputFormatting($outputFormatting = true)

Source: `wire/core/PagesLoader.php`

Set whether loaded pages have their outputFormatting turned on or off

By default, it is turned on.

## Usage

~~~~~
// basic usage
$result = $pagesLoader->setOutputFormatting();

// usage with all arguments
$result = $pagesLoader->setOutputFormatting($outputFormatting = true);
~~~~~

## Arguments

- `$outputFormatting` (optional) `bool`
