# $pagesExportImport->importPageSettings(Page $page, array $settings, array $options)

Source: `wire/core/PagesExportImport.php`

Import native page settings

## Usage

~~~~~
// basic usage
$result = $pagesExportImport->importPageSettings($page, $settings, $options);

// usage with all arguments
$result = $pagesExportImport->importPageSettings(Page $page, array $settings, array $options);
~~~~~

## Arguments

- `$page` `Page`
- `$settings` `array` Contents of the import data 'settings' array
- `$options` `array`
