# $imageSizerEngineGD->processRotate($srcFilename, $dstFilename, $degrees): bool

Source: `wire/core/ImageSizerEngineGD.php`

Process rotate of an image

## Usage

~~~~~
// basic usage
$bool = $imageSizerEngineGD->processRotate($srcFilename, $dstFilename, $degrees);
~~~~~

## Arguments

- `$srcFilename` `string`
- `$dstFilename` `string`
- `$degrees` `int` Clockwise degrees, i.e. 90, 180, 270, -90, -180, -270

## Return value

- `bool`
