# $wireDataDB->load($name): array|mixed|null

Source: `wire/core/WireDataDB.php`

Load a value or all values

## Usage

~~~~~
// basic usage
$array = $wireDataDB->load($name);
~~~~~

## Arguments

- `$name` `string|bool` Property name to load or boolean true to load all

## Return value

- `array|mixed|null`

## Exceptions

- `WireException`
