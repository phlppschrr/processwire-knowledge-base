# $languages->translator(Language $language): LanguageTranslator

Source: `wire/modules/LanguageSupport/Languages.php`

Return the LanguageTranslator instance for the given language

## Usage

~~~~~
// basic usage
$languageTranslator = $languages->translator($language);

// usage with all arguments
$languageTranslator = $languages->translator(Language $language);
~~~~~

## Arguments

- `$language` `Language`

## Return value

- `LanguageTranslator`
