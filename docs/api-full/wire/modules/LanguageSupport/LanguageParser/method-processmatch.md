# $languageParser->processMatch(array $match)

Source: `wire/modules/LanguageSupport/LanguageParser.php`

Process the match and populate $this->untranslated and $this->comments

## Usage

~~~~~
// basic usage
$result = $languageParser->processMatch($match);

// usage with all arguments
$result = $languageParser->processMatch(array $match);
~~~~~

## Arguments

- `$match` `array`
