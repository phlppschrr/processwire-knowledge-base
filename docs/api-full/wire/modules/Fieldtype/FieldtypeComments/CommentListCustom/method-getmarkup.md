# $commentListCustom->getMarkup($property = ''): array|mixed|null

Source: `wire/modules/Fieldtype/FieldtypeComments/CommentListCustom.php`

Get markup property or all markup

## Usage

~~~~~
// basic usage
$array = $commentListCustom->getMarkup();

// usage with all arguments
$array = $commentListCustom->getMarkup($property = '');
~~~~~

## Arguments

- `$property` (optional) `string` Specify property or omit to get all

## Return value

- `array|mixed|null`
