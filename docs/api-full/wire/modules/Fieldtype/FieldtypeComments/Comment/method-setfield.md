# $comment->setField(Field $field)

Source: `wire/modules/Fieldtype/FieldtypeComments/Comment.php`

Set Field that this Comment belongs to

## Usage

~~~~~
// basic usage
$result = $comment->setField($field);

// usage with all arguments
$result = $comment->setField(Field $field);
~~~~~

## Arguments

- `$field` `Field`
