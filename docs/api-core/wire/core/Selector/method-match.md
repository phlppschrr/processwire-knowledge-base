# $selector->match($value1, $value2): bool

Source: `wire/core/Selector.php`

Does $value1 match $value2?

## Usage

~~~~~
// basic usage
$bool = $selector->match($value1, $value2);
~~~~~

## Arguments

- `$value1` `mixed` Dynamic comparison value
- `$value2` `string` User-supplied value to compare against

## Return value

- `bool`
