# $pagesLoader->getNumChildren($page): int

Source: `wire/core/PagesLoader.php`

Load total number of children from DB for given page

## Usage

~~~~~
// basic usage
$int = $pagesLoader->getNumChildren($page);
~~~~~

## Arguments

- `$page` `int|Page` Page or Page ID

## Return value

- `int`

## Exceptions

- `WireException`

3.0.172
