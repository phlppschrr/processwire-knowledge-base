# $imageSizerEngine->setFlip($flip): self

Source: `wire/core/ImageSizerEngine.php`

Set flip

Specify one of: 'vertical' or 'horizontal', also accepts
shorter versions like, 'vert', 'horiz', 'v', 'h', etc.

## Usage

~~~~~
// basic usage
$result = $imageSizerEngine->setFlip($flip);
~~~~~

## Arguments

- $flip

## Return value

- `self`
