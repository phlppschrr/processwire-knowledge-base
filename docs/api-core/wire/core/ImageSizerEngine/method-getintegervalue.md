# $imageSizerEngine->getIntegerValue($n, $min, $max): int

Source: `wire/core/ImageSizerEngine.php`

Get integer value within given range

## Usage

~~~~~
// basic usage
$int = $imageSizerEngine->getIntegerValue($n, $min, $max);
~~~~~

## Arguments

- `$n` `int` Number to require in given range
- `$min` `int` Minimum allowed number
- `$max` `int` Maximum allowed number

## Return value

- `int`
