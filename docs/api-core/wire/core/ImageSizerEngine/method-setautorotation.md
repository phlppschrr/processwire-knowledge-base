# $imageSizerEngine->setAutoRotation($value = true): self

Source: `wire/core/ImageSizerEngine.php`

Turn on/off auto rotation

## Usage

~~~~~
// basic usage
$result = $imageSizerEngine->setAutoRotation();

// usage with all arguments
$result = $imageSizerEngine->setAutoRotation($value = true);
~~~~~

## Arguments

- `$value` (optional) `bool` Whether to auto-rotate or not (default = true)

## Return value

- `self`
