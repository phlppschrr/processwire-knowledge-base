# $inputfieldWrapper->find($selector): InputfieldsArray

Source: `wire/core/InputfieldWrapper.php`

Find all children Inputfields matching a selector string

## Usage

~~~~~
// basic usage
$inputfieldsArray = $inputfieldWrapper->find($selector);
~~~~~

## Arguments

- `$selector` `string` Required selector string to filter the children by

## Return value

- `InputfieldsArray`
