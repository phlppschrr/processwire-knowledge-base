# $page->descendant($selector = '', array $options = array()): Page|NullPage

Source: `wire/core/Page.php`

Find one descendant page, alias of `Page::findOne()`, see that method for details. @since 3.0.116

## Usage

~~~~~
// basic usage
$page = $page->descendant();

// usage with all arguments
$page = $page->descendant($selector = '', array $options = array());
~~~~~
