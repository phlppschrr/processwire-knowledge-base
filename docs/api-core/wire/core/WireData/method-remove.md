# $wireData->remove($key): $this

Source: `wire/core/WireData.php`

Remove a previously set property

## Example

~~~~~
$item->remove('some_property');
~~~~~

## Usage

~~~~~
// basic usage
$result = $wireData->remove($key);
~~~~~

## Arguments

- `$key` `string` Name of property you want to remove

## Return value

- `$this`
