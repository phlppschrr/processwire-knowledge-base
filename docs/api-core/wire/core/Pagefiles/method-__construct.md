# $pagefiles->__construct(Page $page)

Source: `wire/core/Pagefiles.php`

Construct a Pagefiles object

## Usage

~~~~~
// basic usage
$result = $pagefiles->__construct($page);

// usage with all arguments
$result = $pagefiles->__construct(Page $page);
~~~~~

## Arguments

- `$page` `Page` The page associated with this Pagefiles instance
