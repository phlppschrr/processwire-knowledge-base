# $templateFile->__construct($filename = '')

Source: `wire/core/TemplateFile.php`

Construct the template file

## Usage

~~~~~
// basic usage
$result = $templateFile->__construct();

// usage with all arguments
$result = $templateFile->__construct($filename = '');
~~~~~

## Arguments

- `$filename` (optional) `string` Full path and filename to the PHP template file
