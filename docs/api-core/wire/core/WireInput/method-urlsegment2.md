# $wireInput->urlSegment2($get = ''): string|int|bool

Source: `wire/core/WireInput.php`

Same as urlSegment() method but apply only to 2nd URL segment. (since 3.0.155)

## Usage

~~~~~
// basic usage
$string = $wireInput->urlSegment2();

// usage with all arguments
$string = $wireInput->urlSegment2($get = '');
~~~~~
