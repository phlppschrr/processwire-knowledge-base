# $wireInput->urlSegment3($get = ''): string|int|bool

Source: `wire/core/WireInput.php`

Same as urlSegment() method but apply only to 3rd URL segment. (since 3.0.155)

## Usage

~~~~~
// basic usage
$string = $wireInput->urlSegment3();

// usage with all arguments
$string = $wireInput->urlSegment3($get = '');
~~~~~
