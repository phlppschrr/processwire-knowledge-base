# $pagesRequest->setRedirectUrl($redirectUrl, $type = 301)

Source: `wire/core/PagesRequest.php`

Set the redirect URL

## Usage

~~~~~
// basic usage
$result = $pagesRequest->setRedirectUrl($redirectUrl);

// usage with all arguments
$result = $pagesRequest->setRedirectUrl($redirectUrl, $type = 301);
~~~~~

## Arguments

- `$redirectUrl` `string`
- `$type` (optional) `int`
