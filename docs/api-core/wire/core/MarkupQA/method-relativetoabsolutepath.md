# $markupQA->relativeToAbsolutePath($path): string

Source: `wire/core/MarkupQA.php`

Convert a relative path to be absolute

## Usage

~~~~~
// basic usage
$string = $markupQA->relativeToAbsolutePath($path);
~~~~~

## Arguments

- `$path` `string`

## Return value

- `string` Returns absolute path, or blank string on error
